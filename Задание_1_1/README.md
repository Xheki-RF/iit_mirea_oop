**Постановка задачи**

Разработать программу для анализа температурных данных, полученных сметеостанции за период наблюдения.

Программа должна быть реализована с использованием функционального подхода иобеспечивать обработку температурных данных,   заданных в градусах Фаренгейта, споследующим преобразованием в градусы Цельсия и вычислением средней температуры запериод наблюдения.

**Требования к реализации**

1) Реализовать две пользовательские функции (кроме main):
    - функцию с одним целочисленным параметром для преобразованиятемпературы из градусов Фаренгейта в градусы Цельсия и возвращенияполученного вещественного значения;
    - функцию для вычисления и возврата вещественного значения среднейтемпературы, в качестве 2х параметров передается   информация о массиветемператур.
2) Использовать один массив.
3) Запрещено использование сторонних библиотечных функций и динамическойпамяти (new/delete).
5) Формула преобразования: °C = (°F − 32) × (5 / 9).

**Алгоритм работы программы**

1) Ввести кол-во дней наблюдения (n, целое число, 7 ≤ n ≤ 30).
2) Если введенное число не корректно:
    2.1.Вывести на экран сообщение «n is wrong» и перейти к пункту 4.3.
3) Иначе:
    - Создать массив чисел размером n.
        - Ввести значения температуры в Фаренгейтах с клавиатуры,
        - Вычислить значения температуры в градусах Цельсия;
        - Записать полученные значения в массив;
    - С помощью пользовательской функции вычислить среднюютемпературу в градусах Цельсия за период наблюдения.
    - Вывести на экран измененные значения температур в градусах Цельсия и среднюю температуру за период наблюдения в виде, указанном вописании выходных данных.
4) Завершить работу программы.

**Входные данные**

Первая строка: целое число - количество дней наблюдения.
Вторая строка: последовательность целых чисел через пробел - температуры в градусахФаренгейта.

**Выходные данные**

Первая строка: Temperature data: "значения температуры в градусах Цельсия через пробел"
Вторая строка: Average temperature: "значение средней температуры за период наблюдения"
