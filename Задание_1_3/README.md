**Постановка задачи**

Разработать программную систему, демонстрирующую работу с методами класса и управление доступом к данным объекта.

Программная система содержит один класс, включающий:
1) одно открытое свойство целочисленного типа;
2) одно закрытое свойство целочисленного типа.
3) метод установки значения скрытого свойства (название параметра метода должно совпадать с названием свойства);
4) метод получения значения скрытого свойства;
5) метод, изменяющий значения всех свойств объекта:
    - четные значения делятся на 2;
    - нечетные значения умножаются на 2.

**Алгоритм работы программы**

1) Создать объект класса.
2) Ввести два целых числа.
3) Значение первого числа присвоить открытому свойству объекта.
4) Значение второго числа присвоить закрытому свойству объекта.
5) Вызвать метод изменения свойств объекта.
6) Вывести на экран значения открытого и закрытого свойств объекта в заданном формате.
7) При решении задачи необходимо руководствоваться методическим пособием и приложением к методическому пособию

**Входные данные**

Два целых числа через пробел

**Выходные данные**

Первая строка:

Open value: "значение открытого свойства объекта"

Вторая строка:

Hidden value: "значение закрытого свойства объекта"
