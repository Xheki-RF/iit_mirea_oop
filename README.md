Данный репозиторий содержит в себе решенные задачи по предмету "Объектно ориентированное программирование" (ООП) студентов 1-ого курса бакалавриата РТУ МИРЭА по направлению "Программная инженерия".

Для запуска программ, необходимо склонировать данный репозиторий. Для этого в терминале прописываем:
```powershell
git clone https://github.com/Xheki-RF/iit_mirea_oop.git
```
Все дальнейшие инструкции по непосредственному запуску той или иной программы находятся в соответствующих папках.
