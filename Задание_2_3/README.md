**Постановка задачи**

Разработать программную систему на языке C++, моделирующую объект персонажа игры, обладающего именем и внутренними характеристиками. Доступ к данным объекта должен осуществляться с использованием механизма friend.

**Требования к классу Персонажа**

Класс Персонаж должен содержать:

1) три свойства:
    - имя героя (тип строковый);
    - уровень здоровья (целочисленный тип);
    - уровень маны (целочисленный тип);
2) все свойства должны иметь закрытый модификатор доступа;
3) конструктор, инициализирующий значения всех свойств.

Использование публичных методов для получения или изменения значений свойств (get, set) не допускается.

Реализовать функцию вывода исходного состояния объекта, которая:
1) принимает объект класса Персонаж по константной ссылке;
2) выводит на экран имя героя, значение здоровья и значение маны;
3) имеет доступ к закрытым членам класса Персонаж.

Реализовать класс Лекарь, который:
1) содержит метод, изменяющий состояние объекта класса Персонаж:
    - увеличивает значение здоровья на 20;
    - увеличивает значение маны на 10;
2) имеет доступ к закрытым членам класса Персонаж.

**Алгоритм работы программы**

1) Ввести с клавиатуры имя героя, начальное значение здоровья и начальное значение маны.
2) Создать объект класса Персонаж, используя введённые значения.
3) Вызвать функцию для вывода исходного состояния объекта.
4) Создать объект класса Лекарь.
5) Передать объект класса Персонаж в метод класса Лекарь для изменения его состояния.
6) Повторно вызвать функцию для вывода исходного состояния объекта.

**Ограничения**
1) Прямой доступ к закрытым свойствам класса Персонаж вне дружественных сущностей запрещён.
2) Использование публичных геттеров и сеттеров запрещено.
3) Обращение к закрытым членам класса Персонаж из функции main запрещено.
4) Использование механизма friend допускается только в рамках требований задания.

При решении задачи необходимо руководствоваться методическим пособием и приложением к методическому пособию

**Входные данные**
```powershell
Значения через пробел: <Имя> <здоровье> <мана>
```
**Выходные данные**
```powershell
Первая строка: "Hero: <Имя> | Health = <значение здоровья> | Mana = <значение маны>"

Вторая строка: "Hero: <Имя> | Health = <значение здоровья> | Mana = <значение маны>"
```
